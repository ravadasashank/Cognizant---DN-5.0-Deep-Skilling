# myfirstreact - First React Application

A complete React application built for Academic Session 1, demonstrating component creation, JSX rendering, and Single Page Application (SPA) fundamentals.

---

## 🎯 Project Objectives

- Understand the fundamental architecture of React and Single Page Applications (SPAs).
- Create a functional component in React named `App`.
- Render required heading: `"Welcome to the first session of React"`.
- Gain hands-on experience with JSX (JavaScript XML) syntax.
- Understand the build and execution flow using Node package manager (`npm`).

---

## 📋 Prerequisites

Ensure you have the following software installed on your system before proceeding:

1. **Node.js**: v14.0.0 or higher (v18+ recommended)
2. **NPM**: v6.0.0 or higher (comes bundled with Node.js)
3. **Code Editor**: Visual Studio Code (VS Code) or any modern editor

---

## 🚀 Installation Steps

1. **Clone or Download the Project**:
   Ensure all files are placed in the `myfirstreact` folder directory.

2. **Open Terminal / Command Prompt**:
   Navigate to the project root folder:
   ```bash
   cd myfirstreact
   ```

3. **Install Dependencies**:
   Execute the following command to download and install all necessary npm packages (`react`, `react-dom`, `react-scripts`):
   ```bash
   npm install
   ```

---

## ▶️ How to Run the Application

Start the local development server by executing:

```bash
npm start
```

- This command compiles the React code and starts the development server.
- The app will automatically open in your default browser at:
  ```
  http://localhost:3000
  ```

---

## 💻 Expected Output

Upon visiting `http://localhost:3000`, the browser will display:

- **Heading**: `"Welcome to the first session of React"` in bold typography.
- **UI Design**: Dark-mode responsive card layout with glassmorphic backdrop and animated React logo badge.
- **Metadata**: Badges displaying *Functional Component*, *React 18 Concurrent DOM*, *Modern Glassmorphism*, and *Create React App*.
