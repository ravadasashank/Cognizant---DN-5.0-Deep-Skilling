# E-commerce Platform Search Function

A Java console application that implements and compares Linear Search and Binary Search algorithms for searching products in a catalog, evaluating performance differences using Big O Notation.

---

## 📂 Project Structure

```text
search-platform
 ├── pom.xml
 └── src
      └── main
           └── java
                └── com.cognizant.search
                     ├── Main.java
                     ├── Product.java
                     ├── SearchAlgorithms.java
                     └── SearchAnalysis.java
```

---

## ⚙️ How to Run

1. Open a terminal in the project directory:
   ```bash
   cd "Week-1/WEEK 1/search-platform(DSA)"
   ```
2. Build the project using Maven:
   ```bash
   mvn clean package
   ```
3. Run the application:
   ```bash
   mvn exec:java -Dexec.mainClass="com.cognizant.search.Main"
   ```
