# Financial Forecasting Tool

A Core Java command-line application that predicts the future value of an investment using a recursive compound growth algorithm, following the Cognizant FSE Digital Nurture curriculum.

---

## 📂 Project Structure

```text
FinancialForecasting
 └── src
      └── com
           └── cognizant
                └── forecasting
                     ├── FinancialForecast.java
                     └── Main.java
```

---

## ⚙️ How to Compile & Run

Open a terminal or command prompt in the `FinancialForcasting(DSA)` directory:

### 1. Compile the source files:
```bash
javac -d bin src/com/cognizant/forecasting/*.java
```

### 2. Run the program:
```bash
java -cp bin com.cognizant.forecasting.Main
```
